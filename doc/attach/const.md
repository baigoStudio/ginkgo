## 常量参考

----------

#### 预定义常量

| 常量名 | 含义 |
| - | - |
| IN_GINKGO | 是否合法加载 |
| DS | 目录分隔符 |
| GK_VERSION | 框架版本号 |
| GK_PUBLISH | 框架发布时间 |
| GK_NOW | 当前时间 |
| GK_START_TIME | 开始运行时间（时间戳） |
| GK_START_MEM | 开始运行时候的内存占用 |

----------

| 常量名 | 含义 |
