## ginkgo\except

本目录保存异常类。

#### 目录结构

```
project 应用部署目录
+-- ginkgo         框架系统目录
|   +-- except  异常目录
|   |   +-- class_not_found.class.php 类未找到异常
|   |   +-- db_except.class.php       数据库异常
|   |   +-- http_except.class.php     HTTP 异常
|   |   +-- tpl_not_found.class.php   模板未找到异常
|   |   +-- vld_except.class.php      验证器异常
|   |   +--  ...
|   |
|   +--  ...
|
+--  ...
```
